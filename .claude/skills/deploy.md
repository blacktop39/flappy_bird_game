# 배포 프로세스

다음 순서대로 실행하세요:

## 1. 코드 분석
먼저 flutter analyze를 실행하여 코드 품질을 확인합니다.
심각한 에러가 있으면 중단하고 사용자에게 알립니다.

## 2. 웹 빌드
flutter build web --release 명령어로 프로덕션 빌드를 생성합니다.
빌드 시간과 결과를 사용자에게 알립니다.

## 3. 빌드 결과 확인
build/web 디렉토리가 정상적으로 생성되었는지 확인합니다.
주요 파일들(index.html, main.dart.js)이 존재하는지 체크합니다.

## 4. 배포 안내
빌드가 성공하면 다음 정보를 제공합니다:
- 빌드된 파일 위치
- 파일 크기
- GitHub Pages 배포 방법
- 로컬에서 테스트하는 방법

## 5. 완료 메시지
배포 준비가 완료되었음을 알리고 다음 단계를 안내합니다.
